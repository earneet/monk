为 monk 确定**美术方向与执行规范**,统一全资产视觉语言。此前 GDD §10 美术方向暂不定,本指南确定。指南是美术产出的依据(原型期占位 / 正式期资产),不包含实际美术资产。

## 3. 整体基调〔AG1〕

**禅意水墨**:契合 monk 寺院 / 一休扫地主题(禅意、东方)。水墨晕染、大量留白、毛笔笔触。整体「少即是多」,辨识度高。

## 4. 配色

| 角色 | 色值 | 用途 |
|---|---|---|
| 浓墨 | `#2B2B2B` | 边界、角色轮廓、关闭的门 |
| 淡墨 | `#6B6B6B` | 已扫格填充、次要元素 |
| 灰 | `#A0A0A0` | 辅助层次 |
| 宣纸白(留白) | `#F5F0E6` | 底色、未扫格 |
| **朱砂红** | `#C2452D` | 玩家点缀、机关激活、终点、印章(点睛) |
| 青(淡彩) | `#5A7A8A` | 流水 |
| 赭石(淡彩) | `#A0785A` | 植被、木(桥) |

> 原则:墨为主、留白为底、朱砂点睛、淡彩辅助。

## 5. 网格与场景

- **格子**:宣纸纹理底,墨线勾勒边界;**已扫格**=淡墨填充,**未扫格**=宣纸白(墨色深浅直观表达清扫进度)
- **假山**:水墨皴法(山石肌理)
- **流水(静态)**:青色淡彩 + 水墨波纹 / 晕染

## 6. 角色(小和尚)

- 水墨简笔(光头、僧袍),**朱砂点缀**(袈裟红、戒疤)——在墨色背景中突出可辨
- 移动:笔触流动感(扫帚拖出墨痕,呼应「扫地」主题)

## 7. 机制视觉

| 机制 | 视觉 |
|---|---|
| 门 | 水墨门框;关闭=浓墨,开启=淡彩透光 |
| 机关 | **朱砂印章**;踩下=印章「盖」下(朱砂印记) |
| 传送门 | 水墨漩涡 / 晕染涡旋 |
| 桥 | 赭石水墨木板;铺放=实体,未铺=虚影 |
| 动态水 | 涨=深墨(不可通行),落=淡墨 / 白(可通行)——水位以墨色深浅表达 |

## 8. 动画

- **晕染过渡**:格子状态变化(扫过、门开、水涨落)以水墨晕染展开
- **笔触流动**:玩家移动轨迹的墨痕
- **留白呼吸**:背景宣纸微妙起伏(禅意氛围,低强度)

## 9. UI

- **留白界面**:宣纸底,极简
- **毛笔字体**:标题 / 章节名(搭配正文中文字体,保证可读)
- **印章元素**:按钮、章节标记、通关印记
- 整体:禅意、克制

## 10. 资源规范

- **tile 分辨率**:128×128(高分辨率,适配缩放与高 DPI)
- **格式**:PNG(带透明);水墨笔触需抗锯齿、保留晕染层次(避免压缩失真)
- **调色板**:固定(§4),全资产统一取色
- **命名**:`assets/art/<类别>/<名称>.png`(类别:tiles、characters、mechanics、ui、effects)

## 11. 资产来源与阶段策略

禅意水墨美术门槛较高(需专业水墨能力)。分阶段:

- **原型期**(当前):**简单占位**(色块 / 几何线条 + 调色板配色),验证玩法;不投入水墨资产
- **正式期(起步)**:**AI 生成**(水墨风格 prompt)+ **资产库**补充,低成本铺量
- **正式期(精修)**:**关键资产**(角色、标志性场景、BOSS 关)外包专业水墨美术精修
- 一致性由统一调色板 + 风格参考图保证
